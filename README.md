# Desafio OniMusc

## Instruções de execução do código

Pré-requisitos:
- Python 3+ instalado
- Pip instalado

Como executar:

1- Clone o projeto
	git clone https://github.com/leticia-rodrigues2/Desafio.git
	
2- Acesse a raiz do projeto
	cd Desafi
	o

3- Baixe as dependencias
	pip install django
	pip install xlwt
	pip install openpyxl
4- Execute o projeto
	python manage.py runserver
